DEFECT TYPE: Missing Component
CATEGORY: Assembly Defect
IPC CLASSIFICATION: IPC-A-610 Class 1/2/3 — Always Defect

DEFINITION:
A missing component defect occurs when a required electronic component is absent from its designated location on the PCB. This includes fully missing components, components that failed to pick from feeder, or components that were placed but fell off during processing.

VISUAL CHARACTERISTICS:
- Empty pads at designated component location
- Pads may be bare copper or show solder paste / reflowed solder without component
- Sometimes solder tack marks visible where component briefly adhered before displacement
- Pad oxidation or contamination if missing component was undetected over time
- Tombstoned partner component may indicate one end placed, one missing (see tombstoning defect)

TYPES:
1. Complete Missing: Component entirely absent from feeder pick-up or not placed
2. Post-Placement Loss: Component placed but fell off before/during reflow
3. Feeder Empty: Component absent due to feeder running out of parts
4. Skip Placement: Placement machine error caused position to be skipped
5. Wrong Bill of Materials: Component intentionally omitted (BOM error — engineering issue)

ROOT CAUSES:
- Feeder empty or component tape running out mid-batch
- Pick-and-place nozzle clogging or vacuum loss causing drop
- Component falling off board during handling before reflow
- Placement program error or coordinate offset
- Component supply shortage — feeder loaded with wrong or insufficient qty
- Static discharge causing component adhesion failure
- Board warpage preventing reliable vacuum pickup
- Reel splice failures disrupting component feed

ELECTRICAL IMPACT:
- Open circuit at component location
- Function loss of subcircuit (missing decoupling cap → power instability)
- Missing resistor in voltage divider → incorrect reference voltage
- Missing protection components (TVS, fuse) → ESD or overcurrent vulnerability
- Missing filter components → EMI non-compliance
- System-level failure dependent on component criticality

CRITICALITY LEVELS:
- CRITICAL: Missing power regulation component, missing safety protection (fuse, TVS, varistor)
- HIGH: Missing functional IC, missing oscillator/crystal
- MEDIUM: Missing decoupling capacitor, missing pull-up/pull-down resistor
- LOW: Missing test point, missing optional/DNP component (if correctly marked DNP)

NOTE: DNP (Do Not Populate) components marked in BOM/assembly drawing must NOT be flagged as defects.

DETECTION METHODS:
- AOI: Primary method — component presence/absence detection via silhouette and pad coverage
- X-ray: For detecting missing components under packages or in BGA fields
- ICT (In-Circuit Test): Detects electrical absence of functional components
- Functional test: Detects circuit failures caused by missing components
- MDA (Manufacturing Defect Analysis): Comprehensive continuity verification

ACCEPT/REJECT CRITERIA:
- REJECT: Any missing component that is NOT marked DNP in the assembly drawing
- ACCEPT: Components marked DNP (Do Not Populate) in current revision BOM
- REWORK: Place missing component if detected before final QC — requires re-soldering and re-inspection
- REJECT: If missing component cannot be safely placed post-reflow (e.g., BGA)

REWORK PROCEDURE:
1. Verify component reference designator against current assembly revision BOM
2. Confirm component value, package, and polarity per assembly drawing
3. Apply flux to pads, place component with appropriate soldering method
4. Inspect solder joints per IPC-A-610 acceptance criteria
5. Re-run AOI/ICT on reworked location
6. Document rework on traveler/DHR
